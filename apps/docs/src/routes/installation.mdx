# Installation

Get started with Jige UI by installing it in your SolidJS project.

## Package Manager

### npm

```bash
npm install jige-ui
```

### pnpm

```bash
pnpm add jige-ui
```

### yarn

```bash
yarn add jige-ui
```

## Setup

After installation, you'll need to set up the provider and import styles:

```tsx
import { JigeProvider } from "jige-ui";

function App() {
  return <JigeProvider>{/* Your app content */}</JigeProvider>;
}
```

## TypeScript Support

Jige UI is built with TypeScript and includes full type definitions. No additional setup is required for TypeScript projects.
